- tạo 1 project trên github và đẩy lên github
1. tạo 1 github repository (folder) - nơi chứa code
2. clone project này: git clone link_your_repository
3. viết code
4. git status (kiểm tra trạng thái files)
5. git add . (thêm tất cả các files)
6. git commit -m "your_message" (tạo nội dung commit)
7. git push origin main

- tạo project local và đẩy lên github
1. git init
2. git status
3. git add .
4. git commit -m "your_message"
5. git remote add origin link_your_repository
6. git push origin master


- làm việc nhóm 
1. git pull = git fetch + git merge (kết hợp file trong máy và file trên github về)
2. git merge (gộp 2 branch vào)
3. git conflict (xảy ra khi xung đột, không merge được)
4. git checkout branch/commit (chuyển nhánh)
